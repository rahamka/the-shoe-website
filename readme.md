# 👟 The Shoe Company - Modern Footwear Landing Page

A sleek, responsive, and high-performance landing page for a footwear brand, built using **Tailwind CSS** and **Vanilla JavaScript**. This project focuses on modern UI/UX principles, featuring smooth scrolling, interactive navigation, and a mobile-first design.

---

## 🚀 Live Demo

You can view the live website here:  
🔗 **[The Shoe Website - Live Preview](https://rahamka.github.io/the-shoe-website/)**

---

## ✨ Features

- **Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
- **Tailwind CSS:** Styled with utility-first CSS for fast and efficient design.
- **Modern UI:** Clean typography using _Archivo_ and _Poppins_ fonts.
- **Interactive Elements:** Custom mobile navigation menu and smooth scroll effects.
- **Performance Optimized:** Uses modern image formats like `.webp` for faster loading.
- **Icons:** Beautifully integrated icons from the _Iconsax_ library.

---

## 🛠️ Tech Stack

- **Tailwind CSS (CLI):** Advanced styling and layout.
- **JavaScript:** Logic for mobile menu and interactions.
- **Google Fonts:** Archivo font family.
- **HTML5:** Semantic structure for better SEO.

---

## 📁 Project Structure

```text
the-shoe-website/
├── images/             # All website images & backgrounds
├── Iconsax/            # Icon library files
├── index.html          # Main entry point
├── output.css          # Compiled Tailwind CSS
├── input.css           # Tailwind source directives
├── tailwind.config.js  # Tailwind configuration
└── script.js           # JavaScript for interactions
```
